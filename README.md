## 短信转发服务 (SMS Forwarder)

一个自用的短信转发系统，可将安卓手机收到的短信自动转发到网页端显示，方便在手机不在身边时查看验证码。

## 功能特点

- 📱 自动接收并转发手机短信
- 🔢 智能识别和提取验证码
- 🌐 网页端实时查看短信内容
- ⚡ 一键复制验证码功能
- 🔒 简单的访问密码保护
- 📱 响应式设计，支持手机访问

## 技术栈

- **后端**: Python + Flask
- **前端**: HTML + 内联样式 (无外部CSS依赖)
- **部署**: Vercel (Serverless)
- **短信转发**: 安卓APP (SMS Forwarder)
